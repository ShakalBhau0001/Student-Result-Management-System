# Student-Result-Management-System
A full-stack student result management system with authentication, built using Node.js, Express, MongoDB, and a simple frontend interface.

---


## 📁 Project Structure

```bash
student-result-backend/
│
├── config/
│   └── db.js              # MongoDB connection
│
├── middleware/
│   └── auth.js            # Authentication middleware
│
├── models/
│   ├── Admin.js
│   ├── Student.js
│   ├── Subject.js
│   └── Result.js
│
├── routes/
│   ├── auth.js
│   ├── students.js
│   ├── subjects.js
│   └── results.js
│
├── public/
│   └── index.html
│
├── .env
├── LICENCE
├── server.js              # Entry point
├── package.json
├── yarn.lock
└── README.md
```
